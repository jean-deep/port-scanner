¿Cómo funciona el escaneo local 100% de navegador?

Esta herramienta utiliza análisis de respuesta HTTP y técnicas de firma temporal. Al realizar peticiones híbridas de tipo CORS y No-CORS a 127.0.0.1, podemos mapear qué puertos están escuchando de forma activa y predecir de qué tipo de proyecto o framework se trata sin enviar datos al exterior.
